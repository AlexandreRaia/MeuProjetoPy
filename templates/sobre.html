Sobre o Sistema

Este sistema foi desenvolvido para demonstrar a criação de um projeto Flask com um layout moderno utilizando Bootstrap.

O sistema inclui uma página inicial com uma mensagem de boas-vindas e cartões de exemplo, além de uma página de sobre com informações adicionais.

Utilizamos o Flask para criar rotas e renderizar templates, e o Bootstrap para estilizar a interface do usuário.
